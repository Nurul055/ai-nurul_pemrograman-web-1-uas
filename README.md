Tugas Akhir Mata Kuliah Pemrograman Web 1
Aplikasi berbasis web untuk membantu pencatatan keuangan harian, mencakup pemasukan dan pengeluaran dengan sistem autentikasi pengguna.

Proyek ini dikembangkan untuk memenuhi tugas UAS Pemrograman Web 1. Fokus utama aplikasi adalah mengimplementasikan operasi CRUD (Create, Read, Update, Delete) serta sistem keamanan dasar melalui Login Authentication.

Aplikasi ini dibangun menggunakan stack teknologi berikut:
Frontend: HTML5, CSS3, dan Bootstrap (untuk tampilan yang responsif).
Backend: PHP (Native/Prosedural).
Database: MySQL.
Tools: XAMPP (Apache & MySQL).

Fitur Utama
1. Sistem Autentikasi: * Login dan Logout untuk menjaga keamanan data pengguna.
2. Dashboard Keuangan: * Menampilkan ringkasan total saldo, pemasukan, dan pengeluaran.
3. Manajemen Data (CRUD):
Tambah Data: Mencatat transaksi baru.
Lihat Data: Menampilkan riwayat transaksi dalam bentuk tabel.
Edit Data: Mengubah detail transaksi jika ada kesalahan input.
Hapus Data: Menghapus catatan transaksi dari sistem.
